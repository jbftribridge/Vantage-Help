Playlist Overview

Playlists are collections of learning content curated by users. Playlists allow authors to easily share their knowledge with learners in a collection format. Learners can subscribe to playlists created by other users, whether they are peers, leaders, or subject matter experts, and use the suggested training in playlists to develop their own careers.

Note: The Public toggle must be ON in order for playlists to be searchable and visible to other users.

 

  • For information about creating playlists: See Create Playlists.
  • For information about editing playlists: See Editing Playlists.
  • For information about sharing playlists: See Manage Playlists.

 

Playlist Tiles

Each playlist is displayed as a tile. Each tile provides the following information and options:

  • Number of items - This displays the number of learning objects in the playlist
  • Title - This is title of the playlists given by the author.
  • Author - Displays the author if you have subscribed to another person's playlist.
  • Rating - Displays the visual approximation of the rating and the number of ratings received for the playlist.  To see the exact rating you can click the stars to open the rating modal.  To learn more about ratings: See Ratings.
  • Comments - This displays the number of comments received on the playlist.  Click the icon to open the Playlist Details page and leave a comment.  To learn more about comments: See Comments.
  • Public Icon - If icon displayed means that the playlist has been made public
  • Ellipsis - The following options are available for each training tile:
    • Share - This selection opens a modal with several options:
      • The user can click to copy the link, which can be pasted anywhere
      • The user can click to toggle the playlist to either Public or Private.  To learn more about discovering playlists: See Public Playlists.
      • The user can enter keywords which make the playlist more relevent in search results.
    • Add more learning - The user is navigated to the homepage to initiate a search for content.
    • Edit - This will enable you to edit the Title and Description of the playlist.
    • Duplicate - This will replicate the playlist and all of its content.
    • Delete - This will delete the selected playlists.
    • Unsubscribe - This will unsubscribe you from the selected playlist.  The option will only show if you are accessing a playlist that you have subscribed to. 

Playlist Details

From the Playlist Details page, users can view the training inside the playlist, and view metadata about the playlist playlist. The details page can be launched by simply clicking on the playlist tile.

Header

The header of the Playlist Details page displays the Title and Description.  

Additionally there are buttons Add from Vantage Catalogue or Add from your learning. 

  • If Add from Vantage Catalogue button is clicked, the user is navigated to the homepage to initiate a search for content.
  • If Add from your learning button is clicked, the modal to add external content is opened.  To learn more about adding external content: See Your added learning.

In the upper-right corner of the header, options are available. To view the options, click the ... button. The following options may display:

  • Share - Click this button to share to open the share modal
    • The user can click to copy the link, which can be pasted anywhere.
    • The user can click to toggle the playlist to either Public or Private.  To learn more about discovering playlists: See Public Playlists.
    • The user can enter keywords which make the playlist more relevent in search results.
  • Print - click this button to print out a text version of the playlist's contents.
  • Ellipsis:
    • Edit - This will enable you to edit the Title and Description of the playlist.
    • Duplicate - This will replicate the playlist and all of its content.
    • Delete - This will delete the selected playlists.
    • Unsubscribe - If you are viewing another user's playlist then the ellipsis will only have an unsubscribe option.  This will unsubscribe you from the selected playlist. 
  • Copy - If you are viewing another user's playlist then you will have a copy button.  This will copy the contents of the playlist and make the user themselves the author of that playlist.

Playlists Metadata

The metadata column located on the right side  of the Playlist Details page displays the following information:

  • Number of items - This displays the number of learning objects in the playlist
  • Author - Displays the author if you have subscribed to another person's playlist.
  • Rating - Displays the visual approximation of the rating and the number of ratings recieved for the playlist.  Click to open the modal and rate the object.  To learn more about ratings: See Ratings.
  • Comments - This displays the number of comments received on the playlist.  Click the navigate the user to the comments section and leave a comment.  To learn more about comments: See Comments.

Playlist Contents

Each learning object added to the playlist structure displays as a tile.  To learn more about learning objects tile: See Learning Object tiles.

The default order is that the newest item added is added at the end of the current list. The items can be dragged and dropped to reorder the structure by the playlist author.

Comments

This displays the comments left by users.   To learn more about comments: See Comments.

Filters and Sorts

The default sort order of playlists is based on the latest activity within the playlists newest ot oldest.  Users can shose to re-sort the list by latest activity old to new or, by title A-Z or Z-A.

Users can find a search option under change your filters drawer on the left side of the page.  The search will look for words in the title and description of playlists in the view.

Users can also filter by playlist type.  The default view is to display all playlists.  Users can also select to only display their own playlists or subscribed playlists.
