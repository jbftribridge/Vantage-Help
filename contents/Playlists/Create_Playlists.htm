Create Playlists

Users can curate collections of training which can then be made accessible by other users in the system via Discover Playlists. Learners can subscribe to playlists created by other users, whether they are peers, leaders, or subject matter experts, and use the suggested training in playlists to develop their own careers.

To create a new playlist, go to: YOUR LEARNING > YOUR PLAYLISTS and click the ADD NEW PLAYLIST button. To only edit a playlist, click edit from the respective ellipsis tile.

Create New Playlist

To begin, first enter the playlist title and description into Add new playlist modal. Then, click the SAVE button to move on to the next step. If you wish to leave the playlist creation process without creating a playlist, click the Cancel link instead.

Playlists can also be created from any learning object in the system.  find the ellipsis in the tile, click ADD TO PLAYLIST.  Then select the ADD TO NEW PLAYLIST button.  This will open a modal to enter the title for the playlist.  The learning object from where you took this action to create the new playlist will be automatically added to the playlist.
