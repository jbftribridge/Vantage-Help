Notifications

In app notification have been enabled so that users receive real time feedback regarding content that they own or have an interest in.  Notification types include:

  • User added learning you created has been rated
  • User added learning you created has had a comment added
  • New content has been added to playlist you have subscribed to
  • Users have subscribed to your playlist

 

New notifications will be apparent when the bell in the navigation header shows a red dot on top of it.

Click the bell to open the notifications pane.  Users can preview the notifications in this view.  If multiple notifications are present for the same learning object or playlist those are aggregated together in a number.

Notification Tiles

View contain the following elements:

  • See all - Navigates user to the main notification page.
  • Settings icon - Navigates user to their profile to edit email preferences.  To learn more about using Notification summary email preferences: See Notifcation email.
  • Time since the triggering activity occurred.
  • Display of user (or aggregate count if multiple users) , action and object acted upon.
  • Delete icon - Users can press to remove the notification from the system.
  • Bell icon - Users can press to mute further notifications for this action and object combination. Muted notification are hidden unless unmuted or the view is unfiltered. Ex: You might mute the notification triggered when other users comment on the specific playlist.

Notification Page

This view displays all notifications in a page view.  If the user has an aggregate notification, they can expand the tile to see all the individual users who have taken the specified action on that object.

The user has an option to delete all notifications from the system.

Filters and Sorts

The default sort order of Your added learning is based on the latest activity within the playlists newest ot oldest.  

Users can find a search option under change your filters drawer on the left side of the page.  The search will look for words in the title of the learning object.

User can opt to show muted notifications.

All notification display as a default.  Users can select to filter to only show specific types of notifications:

  • Subscribed playlists
  • Your playlists
  • Your added learning
