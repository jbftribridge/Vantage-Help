Bookmarks

Bookmarks are a collection of learning content  that the user has encountered wants to save for later. The content may not necessarily need to be part of a curated playlist that they have created.  

The bookmarked items can be found under YOUR LEARNING >  BOOKMARKS from the site navigation.

 

Adding and Removing

Learning object tiles across the site now have the bookmark option within the ellipsis menu. This action will be followed by a toast message that the action has been completed.

Learners can remove the bookmarked items via the ellipsis when accessed from the Bookmark page.

Bookmark actions do trigger xAPI statements. You will find ‘added’ and ‘removed’ verbs for the playlist extension with the name “Bookmark”.  To learn more about xAPI: See xAPI.

 

Filters and Sorts

The default sort order of bookmarks is based on the last updated date of the learning object, newest ot oldest.  Users can shose to re-sort the list by latest activity old to new or, by title A-Z or Z-A.

Users can find a search option under change your filters drawer on the left side of the page.  The search will look for words in the title of learning objects in the view
