User added learning

User added learning allows users to capture training taken outside the system, such as articles and content read online, or conferences, external online courses, books, etc. on their learning transcript.

To enter user added learning, go to YOUR LEARNING > YOUR ADDED LEARNING. Then, click the Add learning button.  Alternatively users can also add learning using the bookmarklet.  To learn more about using the bookmarklet tile: See Bookmarklet.

 

Adding user added learning

Click the add Learning button.  This button is featured in several locations:

  • User added learning page
  • Learning record page
  • Playlist details page

The Add new learning modal opens.  The user completes the applicable fields, with the following minimum required:

  • Source URL
  • Title
  • Source Name
  • Duration

If the user intends for the content to show on their learning record they will click the box next to the 'Did you complete this content?' question. 

When the required fields are satisfied the user will click save.

Saving the bookmarklet

First click the Get bookmarklet button. 

Click in the box with the 'Add learning to Vantage' message. 

While still clicking, drag your cursor into your bookmarks.  When the icon changes to plus then you can release and the bookmarklet will be saved within your bookmarks.

Using the bookmarklet

The bookmarklet enables you to save external learning while you are reading content out side of the Vantage, even if you are not logged in.  When you are on another site and click the bookmarklet link, the feature grabs the relevant information from the page and pre-fills the Add new learning form.  The user can make changes or additions to the data that was pre-filled before clicking Save.  The feature authenticates you as a user and then saves the detail to the Your added learning page.

Ellipsis options

In addition to options available throughout the site for learning objects, users will have some additional options on user added learning.  Those are:

  • Archive - When an object is no longer deemed relevent the creator may archive it.  This will hide the object within the default view
  • Copy to your added learning - Option applies when viewing another users added learning.  Copies the learning object as one that you created to the user's own User added learning page
  • Report as broken/inappropriate - Option applies when viewing another users added learning.  This will mark the content as inactive and it will be added to the moderation report for reported learning. To learn more about using the User added learning moderation: See Reported user added learning.

Filters and Sorts

The default sort order of Your added learning is based on the latest activity within the playlists newest ot oldest.  Users can shose to re-sort the list by latest activity old to new or, by title A-Z or Z-A.

Users can find a search option under change your filters drawer on the left side of the page.  The search will look for words in the title and description of playlists in the view.

User can also select to show archived added learning objects.
