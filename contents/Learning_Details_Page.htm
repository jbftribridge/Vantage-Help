Learning Details Page

From the Learning Details page, users can view expanded details about the learning object including metadata  The details page can be launched by clicking the Details option on the ellipsis.

 

Header

The header of the Playlist Details page displays the Thumbnail, Title and Description.  

Additionally specific data values will appear if the learning object has been tagged in the PwC Professional or US CPE fields. 

in the upper-right corner of the header, options are available. To view the options, click the ... button. The following options may display:

  • Launch - This will launch the learning object.  the method of launch depends on the content type and provider
  • Ellipsis:
    • Share - This action opens an email with a link to the learning object.  the email opens in the default email client for the operating system.
    • Add to playlist - This allows the user to either add the item to one of the user's existing playlists or to create a new playlist.  This action will be followed by a toast message that the action has been completed. To learn more about the Add to playlist modal: See Add to playlist.

Learning Object Metadata

The metadata column located on the right side  of the Learning Details page displays the following information:

  • Type: Displays the modality selected for the learning object, such as online class, curriculum, event, etc.
  • Duration - If specified, the length in hours and minutes for the content
  • Provider - Displays the the source of the content.  In the case of Cornerstone content it displays the provider based on the territory provider.
  • Status - If the learning object has a status this will display on the far right end.
  • Metadata Icons - This will display if the metadata of the content is tagged within specific metadata fields for the content.
    • CPE icon
    • Top Pick
    • Proficiency
    • "New" content
  • Device Compatibility - If selected by administrators, this will display if the content is suitable for PC, tablets, and/or Phone devices.
  • Rating - Displays the visual approximation of the rating and the number of ratings recieved for the playlist.  Click to open the modal and rate the object.  To learn more about ratings: See Ratings.
  • Comments - This displays the number of comments received on the playlist.  Click the navigate the user to the comments section and leave a comment.  To learn more about comments: See Comments.

Composition Contents

If the learning object is a composition, each learning object added to the composition structure displays as a tile.  To learn more about learning objects tile: See Learning Object tiles.

The order of the learning objects is determined by the admin who created the composition.

 

Recommended Learning: People are also learning carousel

The carousel recommends content based on the what content other users have taken.  Specifically, the algorithm looks at the learning record of people who have taken the course that you are looking at the details page for.  It analyzes the most popular content that those users have taken and returns the top 10 learning objects. 

The carousel does account for an remove learning content that already appears on your learning record.

The carousel defaults to displaying the Top 10 items for people globally.  There is a dropdown to select and receive results for the Top 10 in your territory.

Comments

This displays the comments left by users.   To learn more about comments: See Comments.
