Making Content Available for Download

Learner are always on the go.  Enabling content for usage while without a continual wi-fi or cellular signal helps them be productive.  The admin portal has now been enabled with a field to designate content for offline usage.  This field is only available at this time for files such as videos, audio, articles and PDFs created in the Admin Portal.

We have added a field to filter and find only content in which this box has been checked.

We have also added related fields to the catalog export

  • Flagged for download – This means the downloadable box has been checked.
  • File Uploaded – This means that the file has been uploaded.  This will have more meaning during the following release in which admins can upload SCORM files for CSOD assets.
  • Available for Download – If both ‘Flagged for Download’ and ‘File Uploaded’ equal TRUE then this value will be TRUE, else FALSE.
